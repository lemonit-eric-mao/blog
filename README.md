# 项目说明

```
:.                        # 根目录
│
├─_config.yml              # Jekyll 配置文件
│
├─Gemfile                  # RubyGem 配置文件，用于管理 Jekyll 依赖
│
├─index.html               # 网站首页
│
├─search.html              # 搜索结果页面
│
├─.github                  # GitHub 特有的目录
│  └─workflows             # GitHub Actions 工作流目录，存放自动化脚本
│
├─_layouts                 # Jekyll 布局文件目录
│  └─default.html          # Jekyll 主题的布局文件（用来替换原主题文件）
│
└─_posts                   # Jekyll 文章目录，存放 Markdown 格式的博客文章（确保目录结构为：`/blog/年/月/文章标题/index.md`）

```
