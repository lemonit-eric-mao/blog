##### 下载的软件存放位置

```shell
mao-siyu@msy-pc:~$ ll /var/cache/apt/archives/

```

##### 安装后软件默认位置

```shell
mao-siyu@msy-pc:~$ ll /usr/share/

```

##### 可执行文件位置

```shell
mao-siyu@msy-pc:~$ ll /usr/bin/

```

##### 配置文件位置

```shell
mao-siyu@msy-pc:~$ ll /etc/

```

##### lib文件位置

```shell
mao-siyu@msy-pc:~$ ll /usr/lib/

```
