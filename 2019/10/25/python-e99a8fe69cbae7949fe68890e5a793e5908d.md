##### 前置条件

- Python 2.7.14
- `Python3` 不支持 `MySQLdb` 模块，需要用 `pymysql` 替代，`pymysql` 兼容 2.7.14  
  `pip install pymysql` 或 `pip3 install pymysql`
- [Python 仓库](https://pypi.org/ "Python 仓库")
