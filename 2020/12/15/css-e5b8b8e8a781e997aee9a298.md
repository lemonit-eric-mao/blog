###### 固定定位当前层居中，做法

```css
div {
            /*当前层大小*/
            width: 80vw;
            height: 80vh;
            /*固定定位*/
            position: fixed;
            top: 50%;
            left: 50%;
            /*当前层居中*/
            margin-top: -40vh;
            margin-left: -40vw;
}

```
