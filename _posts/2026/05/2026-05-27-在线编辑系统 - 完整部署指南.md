# 公文在线编辑系统 · 完整部署指南

## 一、环境说明

| 项目 | 值 |
|---|---|
| 云服务器外网 IP | `221.180.141.96` |
| 云服务器内网 IP | `10.10.0.2` |

### 服务清单

| 服务 | 用途 | 外网访问端口 | 宿主机内部端口 |
|---|---|---|---|
| **RustFS** | S3 兼容对象存储（存文件） | 39000（S3 API）| 9000 |
| | | 39001（Web 控制台） | 9001 |
| **Collabora CODE** | 在线 Office 编辑引擎 | 9980 | 9980 |
| **FastAPI WOPI** | WOPI 协议桥接服务 | 18001 | 8001 |

```

  │                                ├─ 内网 10.10.0.2:9000 ──► RustFS 容器（读写文件）
  │                                │
  │                                └─ 被 Collabora 内网回调

  └─ 221.180.141.96:9980  ──► Collabora CODE 容器
                                   │
                                   └─ 内网 10.10.0.2:8001 ──► FastAPI WOPI（WOPI 回调）
```

## 二、部署步骤

### 第一步：部署 RustFS + Collabora CODE（合并一个 compose 文件）

RustFS 和 Collabora 都是基础设施容器，放在同一个 `docker-compose.yaml` 中统一管理，一条命令启停。

```bash
# 创建统一的基础设施目录并进入
mkdir -p /mnt/data/infra-service && cd /mnt/data/infra-service

# 创建 RustFS 数据目录并配置权限
mkdir -p ./rustfs/data ./rustfs/logs ./rustfs/certs
chmod -R 777 ./rustfs

# 写入 docker-compose.yaml（包含 RustFS + Collabora 两个服务）
cat << 'EOF' > docker-compose.yaml
services:

  # ====== RustFS：S3 兼容对象存储 ======
  rustfs:
    security_opt:
      - "no-new-privileges:true"
    env_file:
      - ./rustfs.env
    image: rustfs/rustfs:latest
    container_name: rustfs-server
    ports:
      - "9000:9000"   # S3 API（外网通过云防火墙 39000→9000 访问）
      - "9001:9001"   # Web 控制台（外网通过云防火墙 39001→9001 访问）
    volumes:
      - ./rustfs/data:/data
      - ./rustfs/logs:/app/logs
      - ./rustfs/certs:/opt/tls
    networks:
      - dev-infra-network
    restart: unless-stopped
    healthcheck:
      test:
        [
          "CMD", "sh", "-c",
          "curl -f http://127.0.0.1:9000/health && curl -f http://127.0.0.1:9001/rustfs/console/health"
        ]
      interval: 30s
      timeout: 10s
      retries: 3
      start_period: 40s

  # ====== Collabora CODE：在线 Office 编辑引擎 ======
  collabora:
    image: collabora/code:25.04.9.4.1
    container_name: dev-collabora
    ports:
      - "9980:9980"
    environment:
      # 允许任意来源的前端调用（测试环境）；生产环境改为实际域名
      - aliasgroup1=http://.*,https://.*
      # 关闭内置 SSL，走纯 HTTP，调试友好
      - DONT_GEN_SSL_CERTS=true
      - extra_params=--o:ssl.enable=false --o:ssl.termination=false
    networks:
      - dev-infra-network
    restart: unless-stopped
    healthcheck:
      test: ["CMD", "curl", "-f", "http://localhost:9980/hosting/discovery"]
      interval: 10s
      timeout: 5s
      retries: 3
      start_period: 10s

networks:
  dev-infra-network:
    driver: bridge
EOF

# 写入 RustFS 环境变量文件
cat << 'EOF' > rustfs.env
# ====== 凭证配置 ======
RUSTFS_ACCESS_KEY=admin_access_key_xyz789
RUSTFS_SECRET_KEY=admin_secret_key_pro456file

# ====== 存储卷配置 ======
RUSTFS_VOLUMES=/data/rustfs{0..3}
RUSTFS_UNSAFE_BYPASS_DISK_CHECK=true

# ====== 网络与服务监听 ======
RUSTFS_ADDRESS=0.0.0.0:9000
RUSTFS_CONSOLE_ENABLE=true
RUSTFS_CONSOLE_ADDRESS=0.0.0.0:9001
RUSTFS_CONSOLE_CORS_ALLOWED_ORIGINS=*

# ====== 日志级别 ======
RUSTFS_OBS_LOGGER_LEVEL=info
EOF

# 一键启动两个服务
docker compose up -d
```

**验证启动状态：**

```bash
docker compose ps
```

期望输出（两个服务都为 healthy）：

```
NAME            STATUS                    PORTS
rustfs-server   Up X minutes (healthy)    0.0.0.0:9000-9001->9000-9001/tcp
dev-collabora   Up X minutes (healthy)    0.0.0.0:9980->9980/tcp
```

**登录 RustFS 控制台，创建存储桶：**

> 地址：`http://221.180.141.96:39001/`（外网）或 `http://10.10.0.2:9001/`（内网）
>
> 用户名：`admin_access_key_xyz789`
>
> 密码：`admin_secret_key_pro456file`

登录后创建名为 `official-documents` 的存储桶，并上传待测试的文件（`.xls`、`.docx` 等）。

### 第二步：部署 FastAPI WOPI 服务（直接运行在宿主机）

```bash
# 创建目录并进入
mkdir -p /mnt/data/wopi-service && cd /mnt/data/wopi-service
```

将以下内容写入 `wopi_fastapi.py`：

```bash
cat << 'PYEOF' > wopi_fastapi.py
import io
import mimetypes
import boto3
from fastapi import FastAPI, HTTPException, Request
from fastapi.middleware.cors import CORSMiddleware
from fastapi.responses import HTMLResponse, StreamingResponse
from pydantic_settings import BaseSettings, SettingsConfigDict

class Settings(BaseSettings):
    # FastAPI 与 RustFS 同机，走内网内部端口，无需绕外网
    rustfs_endpoint: str = "http://10.10.0.2:9000"
    rustfs_access_key: str = "admin_access_key_xyz789"
    rustfs_secret_key: str = "admin_secret_key_pro456file"
    rustfs_bucket_name: str = "official-documents"

    # 浏览器访问 Collabora 的外网地址
    collabora_public_host: str = "http://221.180.141.96:9980"

    # Collabora 容器回调 WOPI 接口的内网地址（同机走内网）
    fastapi_internal_host: str = "http://10.10.0.2:8001"

    model_config = SettingsConfigDict(
        env_file=".env",
        env_file_encoding="utf-8",
        extra="ignore"
    )

settings = Settings()

app = FastAPI(title="公文系统 WOPI 联调服务")

app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

s3_client = boto3.client(
    's3',
    endpoint_url=settings.rustfs_endpoint,
    aws_access_key_id=settings.rustfs_access_key,
    aws_secret_access_key=settings.rustfs_secret_key
)

@app.get("/", response_class=HTMLResponse)
async def index():
    html_content = f"""
    <!DOCTYPE html>
    <html lang="zh-CN">
    <head>
        <meta charset="UTF-8">
        <title>公文在线编辑 - 联调控制台</title>
        <style>
            body, html {{ margin: 0; padding: 0; width: 100%; height: 100%; font-family: sans-serif; }}
            .control-panel {{ padding: 15px; background: #f5f5f5; border-bottom: 1px solid #ddd; display: flex; gap: 10px; align-items: center; }}
            input {{ padding: 8px; width: 450px; border: 1px solid #ccc; border-radius: 4px; }}
            button {{ padding: 8px 15px; background: #28a745; color: white; border: none; border-radius: 4px; cursor: pointer; font-weight: bold; }}
            button:hover {{ background: #218838; }}
            .iframe-container {{ width: 100%; height: calc(100vh - 70px); background: #eee; }}
            iframe {{ width: 100%; height: 100%; border: none; }}
        </style>
    </head>
    <body>
        <div class="control-panel">
            <label><b>RustFS 文件名：</b></label>
            <input type="text" id="fileIdInput" placeholder="请输入文件名，带后缀，如 test.docx">
            <button onclick="loadDocument()">挂载并预览公文</button>
        </div>
        <div class="iframe-container" id="container">
            <div style="padding: 40px; text-align: center; color: #666;">
                服务就绪。输入 RustFS 中的文件名，点击按钮加载。
            </div>
        </div>
        <script>
            function loadDocument() {{
                var fileId = document.getElementById('fileIdInput').value.trim();
                if (!fileId) {{ alert('请输入文件名！'); return; }}
                var wopiSrc = "{settings.fastapi_internal_host}/wopi/files/" + encodeURIComponent(fileId);
                var finalUrl = "{settings.collabora_public_host}/browser/dist/cool.html?WOPISrc=" + wopiSrc + "&access_token=MOCK_TOKEN&lang=zh-CN";
                document.getElementById('container').innerHTML = '<iframe src="' + finalUrl + '" allowfullscreen></iframe>';
            }}
        </script>
    </body>
    </html>
    """
    return html_content

@app.get("/wopi/files/{file_id}")
async def check_file_info(file_id: str):
    try:
        response = s3_client.head_object(Bucket=settings.rustfs_bucket_name, Key=file_id)
        file_size = response.get('ContentLength', 0)
    except Exception:
        raise HTTPException(status_code=404, detail=f"RustFS [{settings.rustfs_bucket_name}] 中未找到: {file_id}")
    return {
        "BaseFileName": file_id.split('/')[-1],
        "Size": file_size,
        "OwnerId": "admin",
        "UserCanWrite": True,
        "PostMessageOrigin": "*"
    }

@app.get("/wopi/files/{file_id}/contents")
async def get_file_contents(file_id: str):
    try:
        s3_object = s3_client.get_object(Bucket=settings.rustfs_bucket_name, Key=file_id)
        file_data = s3_object['Body'].read()
        content_type, _ = mimetypes.guess_type(file_id)
        if not content_type:
            content_type = "application/vnd.ms-excel" if file_id.endswith('.xls') else "application/octet-stream"
        return StreamingResponse(io.BytesIO(file_data), media_type=content_type)
    except Exception as e:
        raise HTTPException(status_code=500, detail=str(e))

@app.post("/wopi/files/{file_id}/contents")
async def put_file_contents(file_id: str, request: Request):
    try:
        file_data = await request.body()
        if not file_data:
            raise HTTPException(status_code=400, detail="请求体为空")
        content_type, _ = mimetypes.guess_type(file_id)
        if not content_type:
            content_type = "application/octet-stream"
        s3_client.put_object(
            Bucket=settings.rustfs_bucket_name,
            Key=file_id,
            Body=file_data,
            ContentType=content_type
        )
        return {"status": "ok"}
    except HTTPException:
        raise
    except Exception as e:
        raise HTTPException(status_code=500, detail=f"保存失败: {str(e)}")

if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="0.0.0.0", port=8001)
PYEOF
```

## 三、云服务器防火墙端口开放确认

确保云控制台的**安全组/防火墙规则**中已开放以下入站端口：

| 端口 | 协议 | 用途 |
|---|---|---|
| 18001 | TCP | FastAPI WOPI 服务（外网访问） |
| 9980 | TCP | Collabora CODE 编辑器 |
| 39000 | TCP | RustFS S3 API（外网访问，映射到宿主机 9000） |
| 39001 | TCP | RustFS Web 控制台（外网访问，映射到宿主机 9001） |
