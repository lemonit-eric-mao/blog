### 使用typeof 出现的弊端

```
<pre class="line-numbers prism-highlight" data-start="1">```javascript
typeof {} === 'object' // 对的
true
typeof [] === 'object' // 对的
true
typeof null === 'object' // 这个就不对了
true

```
```

### 精确的使用方式

```
<pre class="line-numbers prism-highlight" data-start="1">```javascript
Object.prototype.toString.call({});
"[object Object]"

Object.prototype.toString.call('');
"[object String]"

Object.prototype.toString.call(null);
"[object Null]"

```
